# Environment / Setup Notes

## Requirements
- Python 3.11+
- Ollama installed + running locally
- At least one model pulled (example: `llama3.1:8b`, `qwen2.5:14b`)
- Playwright browser installed: `python -m playwright install chromium`

## First run
```bash
opengrasp setup
opengrasp doctor
```

## Scanner config
- Copy `portals.example.yml` → `portals.yml` and edit.
